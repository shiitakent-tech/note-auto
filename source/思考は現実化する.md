# 資料タイトル / 著者
『思考は現実化する』（原題: Think and Grow Rich） / ナポレオン・ヒル

# この本の一言要約
「人生は自分自身がイメージした通りになる」という核心的な考えに基づき、500人以上の成功者の共通項を20年かけて体系化した、世界で1億部を超える成功哲学の原典。

# 主要な要点

- 願望実現のための「6か条」を実践する
  - 要点: 漠然とした願いではなく、具体的金額・期限・差し出す代償・詳細な計画を紙に書き、毎日2回音読して潜在意識に刷り込む
  - 出典: 第2章「願望」
  - 補足: 鉄鋼王カーネギーがヒルに授けた秘伝とされる。起床直後と就寝直前に、既に実現したかのように実感しながら大声で読む

- 明確な目標（願望）の設定がすべての起点
  - 要点: 成功への第一歩は「燃えるような強い意欲」を持つこと
  - 出典: 第2章「願望」
  - 補足: 「お金持ちになりたい」ではなく「3年以内に年収1000万円」など具体的で測定可能な目標が必要

- 揺るぎない「信念」を持つ
  - 要点: 願望が必ず実現すると心から信じる精神状態が、思考を現実化させる鍵
  - 出典: 第4章「自己暗示」／成功法則第4条
  - 補足: 強い信念は、一時的な挫折や困難でも心が折れず行動を継続する原動力

- マスターマインドの力を活用する
  - 要点: 二人以上が調和の精神で集まると、個人の能力を超えた知的な相乗効果が生まれる
  - 出典: 第10章「マスターマインド」
  - 補足: 信頼できる仲間と定期的に集まりアイデアを共有することで成長が加速

- 「報酬以上の働き」を習慣にする
  - 要点: 受け取る報酬以上の価値を提供し続けると、結果的に大きな利益やチャンスが殺到する
  - 出典: 第7章「計画的な行動」／成功法則第1条
  - 補足: 期待以上の価値を提供する人に仕事が集中するのと同様、時間差で必ず報われる

- 速やかな「決断」を下す
  - 要点: 成功者は決断が速く一度の決断を変えない。失敗する人は決断が遅くすぐ翻す
  - 出典: 第8章「決断」
  - 補足: 25,000人以上の調査で、失敗の最大原因は「決断力の不足」

- 「忍耐力」を身につけ努力を習慣化する
  - 要点: 結果が出ない時期や逆境でも、目標達成まであきらめず継続する忍耐が不可欠
  - 出典: 第9章「忍耐」
  - 補足: 毎日決まった時間に努力し2ヶ月継続すれば、歯磨きのように「やらないと気持ち悪い」習慣レベルになる

# 読者がすぐ試せる実践ポイント
- 目標の明確化と音読 / ノートにSMARTな目標を3つ書き、朝晩2回音読する（手で書き音読すると脳が重要情報と認識し、潜在意識がチャンスを探し始める）
- ポジティブ情報の強制入力 / 毎日15分以上、成功者の動画や名著などポジティブな情報に触れ思考の基準値を上げる（プライミング効果）
- フィードバック環境の構築 / 目標を信頼できる第三者に発表し相互フィードバックをもらう（ただ考えるだけは達成率43%、計画を作り第三者に進捗報告したグループは76%）

- 「明確な目標を持ち、それを達成したいという強い欲望を持つことが成功への第一歩だ」
