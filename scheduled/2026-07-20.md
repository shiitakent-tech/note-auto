面接が終わった帰り道の、あの独特の重さ。わかってくれる人、いますよね。「言いたかったのは、そういうことじゃなかったのに」——うまく話せなかった後悔だけが、ずっと残る。

いちばんつらかったのは、**"練習のしようがない"**ことでした。

## なぜ"ひとり練習"では、面接は上達しないのか

先にひとつだけ、大事な話を。ここが腹落ちすると、後の練習が全部効いてきます。

参考書を読むだけで面接が上達しない理由は、はっきりしています。面接の本当の勝負は、**「用意した答え」ではなく「想定外の深掘り」への対応**で決まるからです。

「なぜ転職を？」——ここまでは誰でも用意できる。でも本番の面接官は、そのあとに「じゃあ、今の会社でその不満は解決できなかったんですか？」と重ねてくる。この二の矢、三の矢で、多くの人は崩れます。

だから練習で必要なのは、**予測できない角度から質問を重ねてくる相手**と、**終わった後の冷静なフィードバック**。この2つ。そして、この2つこそ、AIがいちばん得意とするものなんです。

## Step 1：AIを「面接官」に変える、基本のプロンプト

まずは、ChatGPTを面接官モードに切り替えます。以下をコピペして、【　】を自分の状況に置き換えるだけです。

```
あなたはプロの採用面接官です。これから私の模擬面接を行ってください。

# 私が受ける求人
- 職種：【例）Webマーケティング担当】
- 企業タイプ：【例）BtoBの中堅IT企業】
- 私の経歴：【例）広告代理店で3年、SNS運用を担当】

# 面接のルール
- 質問は1つずつ。私が答えたら、その内容を踏まえて次の質問へ。
- 私の回答が浅いと感じたら、遠慮なく深掘りしてください。
- 面接官になりきり、解説や励ましは挟まないでください。
- では、最初の質問からお願いします。
```

ポイントは、**「質問は1つずつ」「深掘りして」「なりきって」**の3つ。これを入れないと、AIは質問を10個まとめて出したり、親切に解説を始めたりして、"本番の緊張感"が消えてしまいます。

## Step 2：二の矢・三の矢に「耐える」練習

基本の面接が回り始めたら、次はわざと自分を追い込みます。回答した後に、こう打ち込んでください。

```
今の私の回答について、面接官として気になった点を、
さらに2つ、続けて質問してください。
```

これで、本番でいちばん怖い「深掘りの連鎖」を、安全な場所で先に体験できます。最初はしどろもどろでいい。**"詰まる場所"を、本番の前に洗い出しておく**のが目的です。私は毎回、自分の「志望動機」がいちばん弱いと、この練習ではっきり気づきました。

## Step 3：終わったら、AIに"採点"してもらう

模擬面接が一区切りついたら、面接官モードを解除して、フィードバックをもらいます。ここが伸びしろの宝庫です。

```
面接官モードを終了します。今の模擬面接を、採用担当者の視点で評価してください。
# 評価してほしいこと
1. 各回答の良かった点・弱かった点
2. 「ここは印象が悪い」と感じた回答トップ3
3. その3つを、もっと伝わる言い方に書き換えるとしたら？
4. 全体を通して、私が本番までに直すべき最優先ポイント1つ
```

**「書き換えるとしたら？」**が効きます。ダメ出しだけだと落ち込んで終わりですが、"正解の言い回し"まで一緒に出してもらえるので、次の練習ですぐ試せる。この往復が、いちばん自分を伸ばしてくれました。

参考までに、私が実際にAIに直してもらった一例を載せます。ここまで変わります。

- ❌ **Before**：「前職では、いろいろな業務を幅広く経験してきました」
- ⭕ **After**：「前職では広告運用を軸に、月200万円の予算を任され、CPAを3割改善しました。この"数字で語る"姿勢を、御社でも活かせると考えています」

言っている事実は同じでも、"具体的な数字"と"応募先へのつなぎ"が入るだけで、伝わり方がまるで別物になる。この違いを、AIは何度でも根気よく教えてくれます。

## Step 3.5：面接の「難易度」を上げてみる

基本に慣れたら、あえて厳しいモードも試しておくと、本番が驚くほど楽に感じます。

```
次の模擬面接は「やや圧迫気味の面接官」でお願いします。
私の回答の曖昧なところを、鋭く突いてください。
（例：「それは本当にあなたの成果ですか？」「主観では？」など）
```

本番で意地悪な質問が来ても、「あ、練習した」と思えれば、もう怖くありません。**いちばん怖い面接官を、先にAIで体験しておく**。これが効きます。

## おまけ：最強の武器「逆質問」もAIに作らせる

面接の最後、「何か質問は？」で固まる人、多いですよね。ここも用意できます。

```
この求人（【職種・企業タイプ】）の面接の最後にする「逆質問」を、
"意欲が伝わる順"に10個提案してください。
給料や休みの話に偏らない、前向きな質問でお願いします。
```

10個もらって、自分が心から聞きたい2〜3個を選ぶだけ。逆質問は、印象を最後にひっくり返せる数少ないチャンスです。

## Step 4：弱かった回答を「言い換えストック」にする

Step 3でもらった書き換え案は、そのまま流さずにメモへ。「志望動機」「退職理由」「逆質問」——このあたりの"鉄板の弱点"だけでも、自分の言葉に直したストックを持っておくと、本番での安心感がまるで違います。AIが出した言葉を、一度自分の口で声に出して、しっくりくる形に微調整するのを忘れずに。

## やりがちな失敗、3つ

- **1回で満足する** — 模擬面接は"場慣れ"のための反復です。同じ質問でも、3回やると答えが見違えます。最低5回は回してください。
- **いい質問ばかりさせる** — わざと「答えにくい質問を」とお願いを。痛いところを本番前に突かれるのが、この練習の価値です。
- **AIの言葉を丸暗記する** — 暗記した答えは、少し角度を変えられると崩れます。覚えるのは"言葉"でなく、"考え方の軸"を。

## おわりに

面接に落ち続けていた頃の私は、「自分はコミュ力がないんだ」と思い込んでいました。

でも、違ったんです。足りなかったのは才能じゃなくて、ただ**"場数"**でした。本番でしか練習できないと思い込んでいたその場数を、AIが、いくらでも、何度でも用意してくれた。それだけで、景色は本当に変わりました。

面接は、才能で突破するものじゃありません。**準備した回数が、そのまま自信になる**種類の勝負です。
